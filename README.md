#How to add physical media to your Jellyfin Library.

1) Download the required directory e.g. DVD or Bluray
2) Copy the directory to your Librarys directory structure.
3) Rename the directory e.g.
from "NAME HERE (YEAR) - DVD" to "About a Boy (2002) - DVD" or "About a Boy (2002)" if you don't want DVD on the end.
4) scan the library and you should get the About a Boy folder.

When you play the file it will show a 1 second clip with "On DVD" on the screen.

Note: The same clip is played in the Bluray directory i.e. "On DVD". Unfortuately I created the clip a long time ago and I'm not fussed about it saying DVD, just that I know I have to go to my DVD collection.
